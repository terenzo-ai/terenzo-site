# terenzo.ai

Source for the Terenzo marketing website. Static HTML, deployed on Cloudflare Pages.

- Pages: `index.html` (home), `ai-audit.html`, `faq.html`, `about.html`.
- No framework and no build step. Edit the HTML and `styles.css` directly.
- Deploy: connected to Cloudflare Pages. Every push to `main` redeploys. `terenzo.ai` points here.

See `CLAUDE.md` for conventions (voice, AI-visibility rules, design tokens).

Business and client work lives in a separate private repo.
